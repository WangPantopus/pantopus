Vacation hold.

Toggle-heavy variant carrying a date-range pair, scope toggles, forwarding, and an emergency contact. The primary action "Save" lives as a top-bar trailing text button (iOS convention). The active-hold secondary frame replaces the date pickers with a sky-gradient status hero and a ledger of items currently held.

top bar52px, chevron-left + centered "Vacation hold". Trailing primary-600 "Save" text button (Frame 1) or neutral "Edit" (Frame 2).
date pairWhen card holds two chevron rows (From / To) showing weekday + date as the value, sub describing what happens. A timeline strip below shows the span with a 13-day pill centered.
scope togglesFour rows: Packages · Mail & flyers · Marketplace pickups · Civic notices. Civic carries an "Always on" caption next to a disabled toggle — communicates the locked floor.
forwardingMaster toggle row; when on, an additional row with a primary-50 map-pin badge holds the forwarding address. Helper explains what counts as "urgent".
emergency contactAvatar-led row with name + relation + masked phone. Helper "We'll call them if a delivery driver flags an issue at your door."
active heroFrame 2 only. Sky → primary-700 gradient card, pulsing dot + "Hold active" pill, big tabular "5 days left", and a 3-cell stats grid (Packages · Mail · Forwarded).
held ledgerReplaces the scope toggles in the active state. Icon + label + sub + tabular count for each scope. Civic notices show "Delivered as normal" so the user sees the locked floor working.
destructiveFrame 2 ends with a detached "End hold early" card. Sub "Mail resumes tomorrow morning." Frame 1 has no destructive — Save is the action.
helpersEvery helper rewrites between frames. Scheduling: civic always delivered, urgent re-routes same day. Active: everything held resumes Jun 9 morning.
