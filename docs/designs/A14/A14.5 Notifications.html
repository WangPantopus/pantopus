FRAME 1 · POPULATED
A real mix
Tasks lean push-only for fast replies and email-only for receipts. Pulse is quiet by default — only mentions break through. Home gets everything for packages and emergencies; security alerts are locked on for at least one channel.
FRAME 2 · PAUSED (SECONDARY)
Paused for 2 hours
User tapped Pause all. Master card is replaced by an amber banner with countdown and Resume pill. Category cards stay visible at half opacity so the configured pattern is still readable underneath — when the timer ends, that's what comes back on.
