Canonical Pulse post detail — single author + body, reactions bar, inline composer, threaded replies. This is the cleanest expression of A10's post detail variant: author chip up top, intent chip flush-right, body + media in the middle, the rest is conversation.

slot.header_pattern

post_author — 44px avatar with verification badge, two-line identity (name + time · locality), intent chip flush-right. Lost & Found uses the red error chip; Ask/Offer/Event/Share/Alert each swap palette.

slot.body_pattern

body_reactions — 15px body copy, 1–4 image grid (here 2 photos + a tagged map snippet), reactions bar (heart / hand / eye + comment count), composer, nested thread.

slot.primary_cta

inline_reply — 40px rounded composer pinned in flow. Empty frame promotes the composer to focused (primary ring, active send) and replaces the thread with a "Be the first to reply" card.
