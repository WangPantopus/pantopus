Pantopus · A13.9 · Form (single screen)

Edit profile

Multi-section variant of the Form archetype. The user's own profile — avatar, cover, identity, skills, languages, and visibility toggles, stacked top-to-bottom. Save lives in a sticky-bottom bar because edits are usually partial and the user scrolls a lot; the bar swaps between a quiet "All saved" affordance and a full Discard + Save dual-button when changes are pending.
