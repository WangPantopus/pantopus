Disambiguate

Field-heavy variant of the Form archetype. The user just scanned a piece of physical mail addressed to "M. Kovács" or similar; OCR returned a name that matches multiple residents at the address. This screen pairs the captured image (with OCR bounding box) against a ranked candidate list, so the recipient can be resolved in one tap. Quick-action chips ("This is me" / "Route to…") sit above the list for the obvious cases.
