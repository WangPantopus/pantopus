Saved places

The list of locations a user has saved, plus the bookmark affordance that creates them. Built on the List-of-Rows archetype (pushed screen, back chevron, no tab bar) and the A11 Explore map + sheet. Identity tokens tint each row by type — Home green, Work violet, Saved sky.
