Pantopus · Mobile Archetype

List of rows

The highest-leverage mobile pattern — 30+ screens inherit from it. One shell, three row templates, one empty state. Subsequent screens only need to fill the slots below.

Archetype slots

Fields that change per screen. Everything else — top bar, spacing, card style, FAB, tab bar hidden — stays consistent across all 30+ inheriting screens.

slot.titleHeader text centered in the top bar (e.g. "Bills", "Documents", "Members").
slot.tabs[]Optional tab strip labels with counts, e.g. ["Upcoming (4)", "Paid (12)", "All (16)"]. Omit for category-grouped lists.
slot.row_templateOne of: status_chip (Bills), file_chevron (Docs), avatar_kebab (Members).
slot.row_fields[]Data per row. Bills: payee, date, amount, status. Docs: name, size, uploaded. Members: name, role, joined, verified.
slot.groupings[]Optional section headers for category-grouped variant (e.g. "LEASE (2)", "INSURANCE (3)").
slot.fab_actionFAB icon + action. Omit for read-only lists. Defaults to plus, primary-600.
slot.trailing_actionTop-bar right action — typically plus, search, or user-plus.
slot.empty.iconLucide icon for 72×72 identity-tinted empty circle.
slot.empty.headlineH3 sentence case — "No bills yet", "No documents yet".
slot.empty.subcopyMuted explainer, one to two short sentences.
slot.empty.ctaPrimary CTA matching the archetype's primary action (same as FAB).
