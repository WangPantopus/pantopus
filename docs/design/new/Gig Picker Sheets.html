Composer picker sheets

The bottom-sheet sub-modals the gig composer fields open. Post Gig V1 ships native controls on purpose (plain <select>, single date field, bare radios, 4-up photo grid) and A12.8 Post a Task specifies none of these as designed sheets — so five of the six pickers had no spec. Each is built here as a rounded-top sheet (--radius-2xl) with grab handle, title, options, and a confirm affordance, over a dimmed composer backdrop.

Five sheets · one frame each
SHEET 1 · DEADLINE
Date picker
Quick chips (Today · Tomorrow · This weekend · Flexible), a month calendar with the selected day in primary, and an optional “by a specific time” row. Apply confirms the date.
SHEET 2 · CANCELLATION POLICY
Radio cards
Flexible · Moderate · Strict as selectable cards with refund rules, Moderate chosen, plus a gentle “most neighbors pick…” nudge. Apply saves it.
SHEET 3 · URGENCY
Toggle + window
An on switch for “Mark as urgent”, a how-soon chip row, and an amber note about the visibility boost and promotion fee.
SHEET 4 · TAGS
Tags input
A focused field with removable tag chips and an add caret, a suggested-tags cloud you tap to add, and a remaining-count helper. Apply adds them.
SHEET 5 · ATTACHMENT SOURCE
Action menu
iOS-style action sheet: Take a photo · Photo library · Choose a file, each with a tinted icon, plus a separate Cancel card. Tap-to-pick, no apply.
