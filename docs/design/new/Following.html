Following — the Beacons you follow.

List-of-Rows archetype, avatar-first variant. The signed-in user's followed Beacons (creator personas), grouped by activity so the ones posting float to the top. Each row taps through to the A21 Public Beacon profile. Pushed sub-route: tab bar hidden, back chevron, centered title.

Inherits the List-of-Rows archetype + beacon-family shell. Same phone bezel, status bar, verified avatar, and palette as A03.2 Beacon Updates. Rows come from GET /me/following; sections split on unreadCount and latestPost recency. Overflow ••• opens the row action sheet (Frame 2 → Frame 3).
Four frames · populated · action sheet · mute sub-step · empty
FRAME 1 · POPULATED
Grouped by activity
New updates (unread badges, incl. a 25+ cap & an Insiders tier pill) · Active (one muted row, bell-off + dimmed) · Quiet (no recent updates). Activity sort selected.
FRAME 2 · ACTION SHEET
Row overflow
••• on Maple Bakery → Mark seen · Mute (with sub-step) · Unfollow (destructive, error red). Context header shows who you're acting on.
FRAME 3 · MUTE SUB-STEP
Mute duration
1 day · 1 week · 30 days · Custom (up to 1 year). Back chevron returns to the action sheet.
FRAME 4 · EMPTY
Nothing followed yet
Identity-tinted radio-tower, "You're not following any Beacons yet", explainer, primary "Discover Beacons", and a nearby-suggestions hint. No sort control.
